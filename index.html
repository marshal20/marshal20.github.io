Madkour Face Detector

A multi-task CNN model with 3 stages in PyTorch to detect face boundary and facial landmarks with 92% accuracy (0.92 mean average precision). Embedded the 3 stages in a pipelined face detector that handles the multi-scale detection and the further refinement of face detection candidates.

Forward ECG

Solving the forward problem of electrocardiography. A surface distribution of Transmembrane Potential (TMP) on the surface of the heart as a source, generated using cellular automaton to simulate the propagation of the activation wave. Then using Boundary Element Method (BEM) to calculate Body Surface Potentials (BSP). In order to train a Neural Network model to inversely calculate TMP from BSP.

Demo

Madkour Digit Sequence Recognizer

A convolutional recurrent neural network architecture (CRNN) with connectionist temporal classification loss (CTC) to detect digits sequence in a variable length image using PyTorch. Implemented a basic web app to demo the model in an interactive way.

Madkour Fractal Renderer

Fractal renderer that supports mandelbrot set, julia set and mandelbulb 3D fractals with any parameters. It also supports multithreaded rendering and GPU rendering which allows for faster rendering. Animation of any value is possible with a straight forward Qt5 GUI editor. You can export the animation as a video or a sequence of images directly.
